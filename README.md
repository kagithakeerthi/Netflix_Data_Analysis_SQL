# Netflix Data Analysis Using SQL

## Project Overview
This project analyzes Netflix content using SQL to identify trends in movies, TV shows, countries, ratings, directors, and content growth over time.

## Tools Used
- MySQL
- SQL Workbench

## Key Insights
- Movies vs TV Shows distribution
- Top content-producing countries
- Most common ratings
- Top directors
- Longest duration movies
- Content growth trends
- Documentary analysis

## Skills Demonstrated
- SQL Queries
- Joins
- Subqueries
- Aggregate Functions
- Data Analysis
- Business Insights
